איך מתקינים ומשתמשים

  1. Install the Tampermonkey browser extension.
  2. Install / update the userscript: daily-english-reader.user.js — Tampermonkey will prompt you to install; click Install (or Update).
  3. Pick a level above (A / B / C) and click Open BBC article to open it on the original BBC site.
  4. The userscript detects which of the 3 selected articles you opened, highlights the vocabulary, and shows the Hebrew sidebar.
  5. Use the Words and Quiz tabs in the sidebar to study and test yourself.

The overlay runs entirely in your browser. The original article page, images, and layout stay intact. No full article text is published by this site — only vocabulary and quiz metadata.
