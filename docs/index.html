איך מתקינים ומשתמשים

  1. Install the Tampermonkey browser extension.
  2. Install / update the userscript: daily-english-reader.user.js — Tampermonkey will prompt you to install; click Install (or Update).
  3. Run the routine (or wait for the scheduled run) so today's latest.json is published.
  4. Open today's BBC article using the button above.
  5. Use the Words and Quiz tabs in the sidebar to study and test yourself.

The overlay runs entirely in your browser. The original article page, images, and layout stay intact.
